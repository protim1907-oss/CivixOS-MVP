# CivixOS MVP

A clean Next.js MVP for CivixOS that you can push to GitHub and deploy on Vercel.

## Includes
- Citizen login and dashboard
- District discussion page with AI moderation overlay
- Representatives page
- Moderator login and dashboard
- Admin login and dashboard
- Clean CSS with no Tailwind setup required

## Local run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Deploy to Vercel
1. Create a new GitHub repo
2. Upload the contents of this folder
3. Import the repo into Vercel
4. Deploy

## Notes
- This is a clean front-end MVP/demo build
- No external database is required for deployment
- Supabase/API integration can be added in the next step
